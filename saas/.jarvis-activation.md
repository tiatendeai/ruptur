# JARVIS — ativação rápida local no projeto `saas`

## Objetivo

Evitar busca longa na próxima rodada.

Este arquivo existe para ser o **primeiro ponto de entrada local** quando o operador pedir algo como:

- `Jarvis`
- `Jarvis ativar`
- `Jarvis Iniciar`
- `Jarvis Start`
- `Modo Full`

## O que ativar significa aqui

Ativar o Jarvis neste projeto **não** significa criar nova gênese.

Significa:

1. reconhecer o pedido como invocação operacional real
2. reconciliar a superfície atual com a sessão oficial existente
3. reutilizar a sessão ativa em `omega` / `ruptur`, quando houver
4. responder já em modo Jarvis, sem fingir runtime inexistente
